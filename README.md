# Basic-Projects

A collection of small, focused projects. Each project lives in its own folder with its
own README, dependencies, and usage instructions.

## Projects

- Calculator: Advanced modular Python calculator with standard, scientific,
  programmer, and unit conversion modes. See Calculator/README.md.

## Getting Started

1. Choose a project folder.
2. Follow the project's README for setup and usage.

Example (Calculator):

```bash
cd Calculator
python main.py
```

## Contributing

- Keep projects self-contained.
- Include a README, requirements (if any), and tests.
- Prefer clear, minimal dependencies.
